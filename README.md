# Правила расстановки ударений

## Базовое правило

**По умолчанию ударение ставится на последний гласный звук в слове.**

## Исключения

Данный файл содержит правила-исключения, когда ударение ставится НЕ на последний гласный.

## Известные ограничения

### Проблема с правилами `1a` и `1b`

В настоящее время правила `1a` и `1b` работают некорректно в следующих случаях:

#### 1. Когда корень слова заканчивается на "окончание"
Правило ошибочно срабатывает, хотя это не окончание, а часть корня.

**Пример:**
- `боҙ` (лед) 

#### 2. Когда к слову с окончанием добавляется ещё одно окончание
При добавлении второго окончания слово может выглядеть так, будто оканчивается на определённую последовательность, и правило ошибочно применяется.

**Пример:**
- `ярҙам` → `ярҙамы` — при добавлении окончания `-ы` создаётся ложное совпадение с правилом `-мы`

## Требуется доработка

Необходимо доработать правила `1a` и `1b` так, чтобы они различали:
- Настоящие окончания
- Части корня, совпадающие с окончаниями
- Сложные случаи с несколькими окончаниями
